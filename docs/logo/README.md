# Logo concepts pro icovazby.cz

Čtyři návrhy, všechny v zelené (`#10b981` / `#059669` / `#065f46`). Soubory:

- `concept-A-network.svg` — Network nodes (5 uzlů)
- `concept-B-links.svg` — Linked rings (2 prsteny)
- `concept-C-tree.svg` — Holding tree (parent → 3 children)
- `concept-D-monogram.svg` — Stylizované „iv" monogram

Otevři soubory v prohlížeči nebo v Figmě pro preview.

## A. Network nodes 🕸️

Centrální zelený uzel s 4 vnějšími dceřinkami, propojené hranami.

**Metafora:** Přímá vizualizace „vazby" = uzly + hrany.
**Plus:** Jasná identita produktu (= co dělá).
**Mínus:** Vypadá generic — používá to každý druhý SaaS s grafem.

## B. Linked rings 🔗

Dva propojené prsteny (jako olympijské, jen 2). Levý tmavší, pravý světlejší, střední překryv ještě tmavší.

**Metafora:** Klasický symbol pro „propojení / spojení".
**Plus:** Jednoduché, ikonické, dobře čitelné i v 16×16 favicon.
**Mínus:** Lehce abstraktní — nemusí být hned jasné že jde o firmy.

## C. Holding tree 🌳

Jeden velký kruh nahoře (parent), 3 čtverce dole (děti), propojené větvemi.

**Metafora:** Hierarchický holding, „rozkrýt holding" je core feature.
**Plus:** Vysvětluje hned co produkt dělá. Memorable.
**Mínus:** Mírně organic feel — možná moc „startupové", ne enterprise-grade.

## D. Monogram „iv" ✨

Zelený zaoblený čtverec s bílým písmenem „i" + „V" uvnitř (pojmenovaný „iv" = **i**ČO **v**azby).

**Metafora:** Brand letterform, jako Vercel/Linear/Notion.
**Plus:** Skvěle funguje jako favicon a app icon. Profesionální.
**Mínus:** Bez kontextu nepoví co produkt dělá (= méně „grant zdarma").

## Doporučení

Pro **enterprise/B2B pozici** (= cílíš na banky, advokáty) → **D Monogram**. Vypadá nejprofesionálněji, profesionální nakladatel.

Pro **maximální „aha" efekt** u novýchnávštěvníků → **A Network** nebo **C Tree**. Obě hned říkají co produkt dělá.

Pro **versatilitu** (favicon + landing hero + business card) → **B Linked rings**. Univerzálně použitelné.
